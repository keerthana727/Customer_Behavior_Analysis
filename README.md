# Customer Behavior Analysis

End-to-end data analytics project analyzing customer purchasing behavior using Python, SQL Server, and Power BI.

## Tools
Python (Pandas), SQL Server, Power BI

## What I Did
- Cleaned and transformed customer data in Python
- Created derived fields such as age groups
- Loaded data into SQL Server and validated using SQL
- Built an interactive Power BI dashboard with KPIs and slicers

## Key KPIs
- Total Customers
- Average Purchase Amount
- Average Review Rating

## Key Insights
- Majority of customers are non-subscribers
- Clothing category generates the highest revenue
- Young adults contribute the most to revenue

## Files
- Customer_Behavior.ipynb – Data cleaning & SQL load
- Customer Behavior Project.sql – SQL queries
- Customer Behavior Dashboard.pbix – Power BI dashboard
